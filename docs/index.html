ğŸ“¡ Project Purpose

Monitoring Objective

This project uses multi-source satellite data and AI analysis to monitor "gray zone" activities in waters around Taiwan in real time. We track "dark vessels" (ships with AIS turned off), anomalous vessel behavior in military exercise zones, and maritime activity patterns potentially linked to military exercises â€” providing an open, transparent data foundation for maritime security research and policy.

Why Focus on Dark Vessels?

Research by Global Fishing Watch shows that many vessels disable their Automatic Identification System (AIS), disappearing from public monitoring systems. These "dark vessels" are often involved in illegal, unreported, and unregulated fishing (IUU fishing) and may also be linked to gray zone military operations.

ğŸ”¬ Research Methodology

Synthetic Aperture Radar (SAR) Detection

This project uses the same core technology as Global Fishing Watch â€” Synthetic Aperture Radar (SAR) satellite imagery analysis. SAR works day and night, in any weather, penetrating cloud cover to detect vessels at sea.

CSIS "Signals in the Swarm" Methodology

The system references the CSIS Signals in the Swarm research framework, establishing multi-layered suspicious vessel identification:

  • Behavioral Threshold: Detecting vessels with abnormally high dwell time in drill zones but rarely present in fishing hotspots
  • AIS Anomaly Detection: Identifying frequent name changes, prolonged AIS silence (>18 hours), or vessel type changes
  • Time-Series Correlation: Granger causality analysis between dark vessel activity and PLA military sorties

Multi-Source Data Integration

The system integrates data from the following sources, updated every 6 hours:

  • Global Fishing Watch API: SAR dark vessel detection, fishing effort data
  • AISStream.io: Real-time AIS vessel tracking
  • PLA Military Sortie Records: Publicly available defense reports on military activity near Taiwan
